📿 安宅祈愿文（每日默念）专属金文 / Mantra

弟子今日肃心于此，步入家宅，焚香设供，净水诚心。
愿此屋内福气常驻，愿宅运平稳，灾厄不侵，六神安定。
若曾有争执、怨气、病气、霉运留于此地，愿今香愿火一一净化，化煞为祥。
愿吾之家人同心、彼此理解、相扶相护，愿家宅中正安稳、贵人常来、阴影远离。
愿吾所居之处光明满布，安稳如山，祥瑞环绕。
愿吾之愿力与天地感应，愿此地成为福聚之所、安身之居、乐和之家。

🧧 推荐供品清单

  • 清水 + 盐
  • 艾草香或天然清香
  • 平安符 / 家人照片 / 供灯一盏
  • 金银纸（小份）、白米一碗

🔸 1. 清水 + 盐 象征意义：清净身心、破煞除晦。 使用方式：清水一杯加入一撮海盐，放置于宅中中央或玄关，仪式后倒于室外。可搭配擦拭门框、桌角净化用。

🔸 2. 艾草香 / 天然清香 象征意义：驱邪避秽、清气平衡。 使用方式：点香一支至三支，从大门顺时针绕屋一圈，心中默念“愿此香气净化此地，安宅定气”。

🔸 3. 平安符 / 家人照片 / 灯 象征意义：守护家人、聚焦能量、照亮空间。 使用方式：将代表家人的照片或平安符放于祭台或供桌，并点亮一盏灯，象征光明常在、家人同心。

🔸 4. 金银纸 / 白米 金银纸象征供养祖灵与宅神，白米象征仓满地稳。 金银纸可于祈愿后焚化，白米摆供一日后洒入花盆或米缸，象征“生生不息”。
