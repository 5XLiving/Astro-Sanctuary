📘 进阶四 第1课：命理报告结构范式

报告的核心结构


一份完整的命理报告，需兼顾逻辑性、专业性与人本温度。
推荐结构如下：
命盘总览
日主个性分析
结构与格局解读
用神与喜忌分析
大运与流年趋势
重点问题回应（如婚姻、事业）
实际建议（行为 / 调整 / 选择）

撰写顺序建议


先从“整体命盘气场”描述 → 建立读者感知
接着解析“命主特质” → 提升认同与信任
再进入“格局结构 + 用神逻辑” → 奠定专业
后段再谈“未来趋势与应对” → 强调实用与引导

语言包装与写法重点


避免传统术语直翻，例如：“伤官见官”可写为“表达力过强导致权威压力”
多使用图像化比喻，如“你是一棵春天的树，需阳光但不能暴晒”
给予尊重与选择空间，不可定命、定死
举例帮助理解，模糊语言中加入生活实例

常见范例段落


你的命盘属“木火通明”格，表示你天生具备创意表达的能量，适合以语言、艺术或策划为职志方向。
当前大运偏财透出，资源活络、机遇增加，但身弱者须注意承载压力，避免操之过急。
感情方面，夫妻宫带冲，代表容易吸引异性，也易感情不稳，建议在流年合适时再做重大决定。

报告类型与服务配套


可依需求拆分为不同模组：
AI基础报告（快速产出、配合五行）
客制化重点报告（婚姻 / 事业 / 流年）
深度命盘指导（整合命理 + 风水 + 建议）
报告不仅是分析，更是咨询、疗愈与陪伴的桥梁。
