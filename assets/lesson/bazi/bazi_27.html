📘 第27课：专看事业专盘技巧

四象看事业基础


八字中看事业，关键四象为：官、印、财、食。
官：职位、权力、社会责任
印：专业、背景、能力来源
财：资源、收入、市场
食伤：才华、表达、产出

四象协调 → 事业顺畅、可持续成长；缺损或冲克 → 职涯易变或中断。

如何判断事业主轴


官为喜 → 适合上班体制、正职、专业发展
财为喜 → 适合创业、业务、市场相关
印为喜 → 适合研究、教育、幕后支持
食伤为喜 → 适合自媒体、创作、表达型事业

判定顺序：
分析日主强弱
判断格局 → 财格 / 官格 / 食伤格等
对照流年大运，看发展节奏

官印配置的成败关键


官印相生 → 贵格，利于学业、公职、正统发展
官杀混杂 → 事业多阻、方向不定，需有印星化解
官星被冲、合 → 表职位变动或体制更替
官星为忌 → 宜避高位，专注实务型或创意型事业

财星结构的创业潜力


食伤生财，有印助身 → 适合内容创造、输出型创业
比劫夺财 → 竞争强，宜走自营路线，避合作争财
财多身弱 → 主压力大，需调养身心方能承担事业
财为用神 → 宜做实业、商贸、顾问、销售型工作

大运流年事业节奏


运走官印 → 升迁、转职、考试运佳
运走财星 → 创业、收入增长、市场转向
运走食伤 → 表现力提升、发表、创作、曝光
运走比劫 → 主竞争、团队重整、人事调整

事业发展 ≠ 一味追财，需看命盘结构与承载能力。
