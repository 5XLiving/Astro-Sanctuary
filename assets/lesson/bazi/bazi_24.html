📘 进阶三 第4课：专看婚姻专盘技巧

💍 从夫妻宫看感情基础

夫妻宫，即日支，为婚姻最核心之处。
● 日支为配偶的象征，代表伴侣类型与互动模式
● 日支受冲、合、刑 → 表示婚姻多变或介入第三者
● 日支为用神 → 婚姻和谐且能助力人生；为忌神 → 多主感情困扰

🌟 配偶星的力量与位置

● 男命看正财偏财（妻星），女命看正官七杀（夫星）
● 配偶星藏于地支 → 主姻缘隐晦，来得慢
● 配偶星透于天干 → 主主动恋爱、感情外显
● 配偶星合化他星 → 容易婚外情、被争夺
● 配偶星为喜 → 多遇良缘；为忌 → 情感波折

🔍 特殊结构的婚姻信号

● 日支坐劫财或比肩 → 夫妻争执，易第三者
● 配偶星被合走 → 感情不专或被夺爱
● 官杀混杂、财多无制 → 感情复杂、多角关系
● 八字无配偶星 → 婚缘薄、迟婚、或需靠合盘补缘

💞 合盘与婚姻匹配技巧

● 男女日柱相合、日支相合 → 缘分强
● 一方为对方的用神 → 有助成长与安定
● 一方克制对方配偶星 → 易破坏彼此感情
● 合盘中刑冲过多 → 易争执、心灵难靠近

合盘≠全合就好，还需看能否互补、互扶。

📎 实战建议与调整

● 不宜强求配偶星旺，而应配合命局结构
● 流年若冲夫妻宫 → 当年关系多变，宜沟通为上
● 婚姻不顺者，可从风水、命名、择日辅助调整
● 遇“破宫破星”者，宜修内功、自我转化，才能等来佳缘
