📘 进阶四 第3课：多类型客户服务建议模组

💞 01. 婚姻焦虑型

● 常见问题：桃花少、婚姻迟、缘分断裂
● 命理重点：夫妻宫、日支、配偶星、流年合冲
● 建议模组：
→ 分析感情节奏（何时易动情、何时慎选择）
→ 提供择偶建议（性格类型、五行互补）
→ 建议专属行动或择日仪式（化解孤寡、启动姻缘）
→ 温和语气为主，避免批判

💼 02. 事业迷茫型

● 常见问题：工作倦怠、方向模糊、频繁转职
● 命理重点：财官印食四象、用神五行、当前大运
● 建议模组：
→ 提供命格对应的职业偏向
→ 分析转职时机与稳定性
→ 强调“先稳定后突破”的节奏
→ 补充：性格适配、发展建议、能量布局方向

🔍 03. 人生方向型

● 常见问题：找不到人生目标、感觉被困、空转焦虑
● 命理重点：格局主轴、日主潜能、大运变化点
● 建议模组：
→ 命盘定位三段人生：成长期 / 黄金期 / 反思期
→ 提出“此生任务”的象征引导语
→ 给予阶段性策略：学习 / 行动 / 放下
→ 可结合灵性引导、祝福语或祈愿方法

👨‍👩‍👧‍👦 04. 家庭关系型

● 常见问题：父母控制、兄弟争财、育儿烦恼
● 命理重点：年柱 / 月柱 / 子女星、财星流通状态
● 建议模组：
→ 分析家庭互动五行结构
→ 提供沟通调解方式或能量风水调整建议
→ 若问子女运：观察时柱与子女星强弱
→ 可结合父母八字共盘分析，提升理解度

🧘 05. 灵性觉察型

● 常见问题：内在困惑、追问使命、灵性焦虑
● 命理重点：比劫、印星、空亡、贵人、化神
● 建议模组：
→ 命盘找寻“灵性主轴”（如印多者适合修行）
→ 提出“人生课题”关键词与转化路径
→ 可加入人类图 / 占星 / 塔罗辅助
→ 结束时给予祝福语或冥想建议，稳固疗愈感
