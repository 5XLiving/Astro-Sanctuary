🧧 太岁包（Tai Sui Protection）

📿 太岁祈愿金文 / Tai Sui Protection Mantra

弟子今日肃心焚香，敬祀流年太岁星君。
若吾今年冲、刑、害、破太岁，愿以此香愿祈求赦化灾星，平安吉顺。
愿太岁垂怜，护我安康，逢凶化吉，避祸呈祥。
若因流年不利而起诸般波动，愿太岁神君明察护佑，转祸为福，解厄为喜。

愿我之身无疾厄，家无忧患，行事无阻，贵人显现。
愿事业稳定、财运通顺、人和事吉，诸事不逆。
愿过去所犯之罪愿忏悔弥补，未来所行之路愿光明平坦。
愿太岁镇星，护宅护运，镇灾守岁，安我全家。

一愿流年无惊扰，二愿家中有祥光，三愿心念常正，行处皆安。
愿愿不空，愿香有感，愿太岁神明常佑吾身。

语音引导 / Protection Audio

🧧 推荐供品

  • 香三支（或艾草香）
  • 红色蜡烛一对
  • 太岁符 / 太岁图像
  • 金银纸一份

🔸 香三支
象征意义：启请太岁星君，传达祈愿，净化气场。
使用方式：点香三炷，诚念金文，对太岁神位方向行礼。

🔸 红色蜡烛一对
象征意义：红色属火，镇煞开运，稳定气场。
使用方式：供于香旁，点燃时专注愿力释放。

🔸 太岁符 / 太岁神像
象征意义：呼请神灵镇守，安抚冲犯之气。
使用方式：可于供桌摆放，亦可随身携带或放入钱包 / 车上。

🔸 金银纸
象征意义：表达敬意与忏悔，消解流年劫煞。
使用方式：仪式后焚化，配合念诵“愿岁君垂怜，解厄安泰”。

🗓️ 推荐仪式时间：
每年正月初八 / 初九 / 正月十五；
或于年头 / 生日 / 节气转折时进行补运仪式。

🙏 仪式流程建议：
准备供品 → 点香点灯 → 默念太岁金文 → 焚化金纸 → 佩戴护符。
仪式后清静一日，可书写“安太岁”愿望纸贴于日记或桌上。
